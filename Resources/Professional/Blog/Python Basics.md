		&lt;div class&#x3D;&quot;kg-card-markdown&quot;&gt;&lt;p&gt;Activating a virtualenv with python: cd into the directory you want it in. Then you type:&lt;/p&gt;
&lt;pre&gt;&lt;code&gt;virtualenv --system-site-packages ~/Desktop/dev/python/tensorflow
&lt;/code&gt;&lt;/pre&gt;
&lt;p&gt;This sets up the virtual env directory. To activate that virtual environment you then type &lt;code&gt;source ./bin/activate&lt;/code&gt;&lt;/p&gt;
&lt;p&gt;Access the python repl by typing &lt;code&gt;python / python3&lt;/code&gt; at the terminal.&lt;/p&gt;
&lt;p&gt;Accessing help with python you simply type&lt;br&gt;
help(&lt;obj&gt;) to get help about a specific topic.&lt;/p&gt;
&lt;p&gt;Conditional statements are written as:&lt;/p&gt;
&lt;pre&gt;&lt;code&gt;if &amp;lt;expression&amp;gt;:
    code to run here
&lt;/code&gt;&lt;/pre&gt;
&lt;p&gt;Note that conditional bodies are indented four spaces.&lt;/p&gt;
